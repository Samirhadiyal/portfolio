Virtual White Board

Virtual White Board is a Java Swing-based drawing application that allows users to sketch freely on a digital canvas using mouse input. Designed as a beginner-friendly project, it demonstrates the fundamentals of Java GUI development, including real-time rendering, mouse event handling, and custom graphics using the Graphics class. The application features a clean and minimal interface, built entirely with core Java libraries and without any external dependencies, making it easy to understand, run, and modify. Ideal for students and aspiring developers, this project serves as a practical introduction to interactive desktop application development and lays the groundwork for enhancements like color selection, brush customization, and canvas saving.
